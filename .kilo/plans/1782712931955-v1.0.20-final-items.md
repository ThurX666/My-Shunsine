## Task 3: Auto-RP `rpperiksa` untuk SAMD di OnPlayerText

**File:** `gamemodes/main.pwn` — function `TryAutoRoleplay`

Tambah trigger:

```pawn
else if(!strcmp(text, "rpperiksa", true))
{
    if(pData[playerid][pFaction] == SAMD)
    {
        new targetid = INVALID_PLAYER_ID;
        foreach(new i : Player) if(i != playerid && NearPlayer(playerid, i, 5.0))
        {
            targetid = i;
            break;
        }
        if(targetid == INVALID_PLAYER_ID) return 0;
        new Float:hp;
        GetPlayerHealth(targetid, hp);
        format(action, sizeof(action), "sedang memeriksa kesehatan %s.", ReturnName2(targetid));
        SendNearbyMessage(playerid, 20.0, 0xC2A2DAFF, "* %s %s", ReturnName2(playerid), action);
        Health_Show(playerid, targetid);
        return 1;
    }
    else return 1;
}
```

**Alur:** SAMD ketik "rpperiksa" di chat → cari player terdekat dalam 5m → RP message + show health dialog. Non-SAMD diabaikan (return 1 biarkan teks masuk chat biasa).

**Edge case:** Tidak ada player dalam 5m → return 0 (teks masuk chat biasa, tidak ada error). Multiple player dalam radius → ambil player pertama dari foreach (urutan koneksi). Untuk inspeksi lebih selektif nanti bisa tambah `/periksa [playerid]` command.
